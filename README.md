<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

Have an updated version of npm installed
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Max-Farthing/social-media.git
   ```
2. Install NPM packages in BOTH client and server folders
   ```sh
   npm install
   ```
